Этот воркфлоу активирует протокол "Nexus Idea Capture" для мгновенного сохранения гипотез, юридических стратегий и архитектурных набросков.

### 1. Как использовать

Введите `/idea <ваша идея>` для быстрой записи.

### 2. Структура хранения

- Все идеи сохраняются в файл `PROJECT/IDEA/LOG.md`.
- Каждая запись получает метку времени и уникальный ID.
- Идеи классифицируются по категориям:
  - **Legal Strategy** — тактические решения по кейсам
  - **OSINT** — идеи по расследованиям
  - **Automation** — улучшения документо-генерации
  - **Architecture** — структурные решения проекта

### 3. Формат записи

При получении идеи агент должен:

1. Создать `PROJECT/IDEA/LOG.md` если файл не существует.
2. Добавить запись в формате:

```markdown
## [IDEA-NNN] <Краткий заголовок>
- **Дата**: YYYY-MM-DD HH:MM
- **Категория**: Legal Strategy | OSINT | Automation | Architecture
- **Описание**: <Полный текст идеи>
- **Статус**: NEW
- **Связанный кейс**: case-NNNN (если применимо)
```

### 4. Анализ (Review)

Агент может проанализировать `LOG.md` и предложить приоритизацию идей по критерию влияния на активные кейсы.
