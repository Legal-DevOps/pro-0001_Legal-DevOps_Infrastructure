---
description: Установка и запуск асинхронного агента Jules
---

Этот ворклоу проверяет наличие Jules CLI, устанавливает его при необходимости и запускает процесс инициализации или выполнения задачи.

### 1. Проверка и установка Jules

// turbo
npm install -g @google/jules

### 2. Авторизация (требуется один раз)

// turbo
jules auth login

### 3. Запуск Jules

После установки и логина используйте команду ниже для постановки задачи:

```bash
jules task "Опишите вашу задачу здесь"
```

Для просмотра состояния текущих задач:

```bash
jules dashboard
```
